2. for文の基本 ― 同じ処理を繰り返す

たとえば「こんにちは」を5回表示したいとき、print() を5行書くのは面倒です。

print("こんにちは")
print("こんにちは")
print("こんにちは")
print("こんにちは")
print("こんにちは")

これが100回だったら？1000回だったら？もちろん手で書くのは現実的ではありません。そこで登場するのが for 文です。

for i in range(5):
    print("こんにちは")

たった2行で5回の繰り返しが実現できます。range(5) は「0から4まで、つまり5回繰り返す」という意味です。

4. range() を使いこなそう

range() には2つの書き方を覚えておけばOKです。

  • range(stop) : 0 から stop-1 まで
  • range(start, stop) : start から stop-1 まで

特に便利なのが range(start, stop) です。「1から数えたい」ときに使います。

5. リスト ― データをまとめて管理する

ここまでは range() を使って数の繰り返しを学びました。次はリストを学びましょう。

リストとは、複数のデータをひとまとめにして管理できるデータ型です。[ ]（角カッコ）で囲み、要素をカンマで区切ります。

fruits = ["りんご", "バナナ", "みかん"]
コードを試してみよう

リストの要素にアクセスするときは [番号] を使います。ここでも0から数えるのがポイントです。

  • fruits[0] → "りんご"（最初の要素）
  • fruits[1] → "バナナ"（2番目の要素）
  • fruits[2] → "みかん"（3番目の要素）
  • len(fruits) → 3（要素の数）
Syntax

リストは Python で最もよく使うデータ構造のひとつ。ユーザー一覧、商品リスト、テストの点数……何でもリストに入れて管理できるよ。

6. リストを for 文で回す

for文の本領発揮です。リストの要素を1つずつ取り出して処理できます。

fruits = ["りんご", "バナナ", "みかん"]
for fruit in fruits:
    print(fruit)

for fruit in fruits: は「fruits リストから1つずつ取り出して fruit に入れる」という意味です。ループが回るたびに、fruit の中身が次の要素に変わります。

変数名は何でもOKですが、リスト名が fruits（複数形）なら、取り出す変数は fruit（単数形）にすると読みやすいですよ。

演習問題 問題 2

リスト numbers = [3, 7, 2, 9, 5] の合計を for 文で計算し、3 + 7 + 2 + 9 + 5 = 26 と出力してください。

答えを見る 
numbers = [3, 7, 2, 9, 5]
total = 0

for num in numbers:
    total = total + num

print(f"3 + 7 + 2 + 9 + 5 = {total}")

最終チャレンジ 問題 3

商品名リストと価格リストを for 文で回して、番号付きリストと合計金額を出力してください。期待される出力は下記の通りです。

--- 買い物リスト ---
1. 牛乳 → 200円
2. 卵 → 300円
3. パン → 150円
合計: 650円
答えを見る 
names = ["牛乳", "卵", "パン"]
prices = [200, 300, 150]
total = 0

print("--- 買い物リスト ---")

for i in range(len(names)):
    print(f"{i + 1}. {names[i]} → {prices[i]}円")
    total = total + prices[i]

print(f"合計: {total}円")
Syntax

range(len(names)) はリストの長さ分だけ繰り返すテクニック。names[i] と prices[i] のように同じ i で2つのリストを同時に回せるのが便利だよ。実務でもよく使うパターン！

まとめ

この記事では、Pythonの繰り返しとリストの基礎を学びました。

  • for i in range(n): で n回の繰り返し

  • リストは [ ] で複数のデータをまとめて管理
  • for item in list: でリストの要素を1つずつ取り出して処理
  • for と if の組み合わせで、条件に合うデータだけを処理

前回の print・変数・if文と合わせて、これでPythonの基礎文法が一通り身についたはずです。「変数にデータを入れて、条件分岐して、繰り返す」——この3つができれば、もう立派にプログラムが書けます。
