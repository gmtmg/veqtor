1. はじめてのプログラム ― print()

プログラミングで最初にやることは、画面に文字を表示することです。Pythonでは print() という関数を使います。

print("Hello, World!")

print() の () の中に表示したい文字を書きます。文字は "（ダブルクォーテーション）か '（シングルクォーテーション）で囲みます。これを文字列（String）と呼びます。

2. 変数とデータ型

変数（へんすう）は、データを入れておく「名前付きの箱」です。

name = "太郎"
age = 20
print(name)
print(age)

= は「等しい」ではなく「右側の値を左側の名前に入れる」という意味です。数学の等号とは違うので注意しましょう。

主なデータ型

  • str（文字列）: "こんにちは" のように " で囲んだもの
  • int（整数）: 42, -7 など小数点のない数
  • float（小数）: 3.14, -0.5 など小数点のある数
  • bool（真偽値）: True か False のどちらか

文字列の連結

文字列の中に変数を埋め込みたいときは f文字列（f-string）が便利です。f"..." の中で {変数名} と書くと、その変数の値が埋め込まれます。

変数名のルール

  • 英字・数字・アンダースコア _ が使える（数字で始めるのはNG）
  • 大文字と小文字は区別される（Name と name は別の変数）
  • Pythonの予約語（if, for, class など）は使えない

3. 比較演算子

2つの値を比べるための記号を比較演算子と呼びます。比較の結果は True（正しい）か False（正しくない）のどちらかになります。

  • == : 等しい
  • != : 等しくない
  • < : より小さい
  • > : より大きい
  • <= : 以下
  • >= : 以上

4. if文の基本

プログラムで「もし〜なら、こうする」という条件分岐を書くのが if 文です。

if 条件:
    条件がTrueのとき実行される処理

Pythonではインデント（字下げ）がとても重要です。if の次の行は半角スペース4つで字下げします。この字下げがないとエラーになります。

if / elif / else

条件を複数チェックしたいときは elif（else ifの略）を使います。どれにも当てはまらない場合は else で処理します。

temperature = 35

if temperature >= 35:
    print("猛暑日です")
elif temperature >= 25:
    print("夏日です")
else:
    print("過ごしやすい気温です")

ポイントは、上から順にチェックされ、最初にTrueになった条件だけが実行されることです。

5. 実践チャレンジ ― 成績判定プログラム

ここまで学んだことの総仕上げです。変数、比較演算子、if/elif/else を組み合わせて、テストの点数から成績を判定するプログラムを作ってみましょう。

判定ルールはこちらです。

  • 90点以上 → A
  • 80点以上 → B
  • 70点以上 → C
  • 60点以上 → D
  • 60点未満 → F
最終チャレンジ 問題 3

上のルールに従って、score の値から成績（A〜F）を出力してください。5パターンの点数で自動判定します。

答えを見る

    print("A")
elif score >= 80:
    print("B")

    print("C")
elif score >= 60:
    print("D")

    print("F")
